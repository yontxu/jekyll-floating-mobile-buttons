Installation 

ember install ember-floating-mobile-buttons

Usage

Define a simple floating button.

{% highlight handlebars %} {% raw %}{{#floating-mobile-buttons}} {{fa-icon "pencil"}} {{/floating-mobile-buttons}}{% endraw %} {% endhighlight %}

You can additionally define child buttons which will be display grouped by the parent button.

{% highlight handlebars %} {% raw %}{{#floating-mobile-buttons position="bottom right"}} {{#floating-mobile-child-buttons label="Add Item"}} {{fa-icon "user"}} {{/floating-mobile-child-buttons}} {{#floating-mobile-child-buttons label="Remove Item"}} {{fa-icon "trash-o"}} {{/floating-mobile-child-buttons}} {{#floating-mobile-child-buttons label="Edit Item"}} {{fa-icon "pencil"}} {{/floating-mobile-child-buttons}} {{/floating-mobile-buttons}}{% endraw %} {% endhighlight %}

As seen in the example above, you can combine it with you own icons.

Properties

Position

The fixed position of the floating button.

The available list por position:

  • bottom right (default)
  • bottom left
  • top right
  • top left

  • ember serve
